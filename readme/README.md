# 価格検索ツール 使い方

---

## ファイル構成

```
作業フォルダ/
├── price_db.db                   ← SQLiteデータベース（初回実行で自動生成）
├── sqlite_price_lookup.py        ← 価格検索本体（VBAから呼ばれる）
├── db_import.py                  ← CSVをDBに取込むスクリプト
├── run_import.bat                ← ダブルクリックで取込実行
├── 使い方.md
└── csv/                          ← CSVを置くフォルダ
    ├── 標準DB.csv
    ├── 見積DB.csv
    └── エイブル価格表DB.csv
```

---

## 初回セットアップ

1. `csv/` フォルダにCSVを入れる
2. `run_import.bat` をダブルクリック
3. VBAモジュール（.bas）をExcelにインポート

以上で完了。

---

## Excelでの使い方（日常）

1. B2セルで検索対象DBを選ぶ（例: `標準DB`）
2. B4以降に製品名を入力
3. マクロ `RunPriceLookupViaPython` を実行
4. 候補が複数ある場合はポップアップが出るので番号または品番CDを入力

**結果はC〜G列に入る**

| 列 | 内容 |
|----|------|
| C  | 価格 |
| D  | 荷姿 / 単位 |
| E  | 結果メモ（参照DB・品番CD・類似度等） |
| F  | 品番CD |
| G  | 正式材料名 |

---

## データを更新したいとき

1. `csv/` フォルダ内の該当CSVを直接編集
2. `run_import.bat` をダブルクリック

特定のソースだけ更新したい場合はコマンドで実行:

```
python db_import.py --db price_db.db --csvdir csv --source 標準DB
```

---

## 新しいDBを追加したいとき

1. `csv/` フォルダに `○○社DB.csv` を置く
2. `run_import.bat` をダブルクリック
3. Excelの B2セルに `○○社DB` と入力して使える

スクリプトの変更は不要。

---

## CSVのフォーマット

| 列名 | 必須 | 備考 |
|------|------|------|
| 品番CD（または 品番コード） | ○ | |
| 材料名（または 品番名） | ○ | |
| 価格 | - | なければ名称のみ取込 |
| 荷姿 / 単位 | - | あれば取込む |
| 備考 | - | あれば取込む |
| その他の列 | - | 無視される |

エンコード: UTF-8（BOMあり）またはCP932（Shift-JIS）

---

## 価格検索の優先順位

B2で選んだDBに価格がない場合、自動でフォールバックする。

```
B2で選んだDB → 標準DB → 見積DB
```

---

## よくあるエラー

| エラー | 原因 | 対処 |
|--------|------|------|
| `DBファイルが見つかりません` | dbPathとDB名が不一致 | VBAの `dbPath` を確認 |
| `Pythonスクリプトが見つかりません` | スクリプトが別フォルダにある | 同じフォルダに配置 |
| `対象行が多すぎます（XXXX行）` | B列に余分なデータが残っている | 該当セルを選択してDelete |
| `登録済みのDB: ...` エラー | B2のDB名がDBに存在しない | csv/にCSVを置いて取込を実行 |
| `候補なし` | 名称DBに未登録の製品名 | CSVに追記して再取込 |
